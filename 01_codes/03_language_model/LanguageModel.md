# Part 3. 언어 모델
- 언어 모델 : 단어 시퀀스(문장)에 확률을 할당하는 모델
어떤 문장이 있을 때 기계가 이 문장의 적절성을 판단하면, 자연어 처리 성능이 뛰어나다고 할 수 있음

3장에서는 통계에 기반한 전통적 언어 모델(SLM)을 공부한다.
</br>
통계 기반 언어 모델은 실제 자연어와 근사하기 어려움이 있어 인공 신경망으로 대체되었지만, 방법론을 배우는 것도 중요하다.

## 03-1. 언어 모델이란?
언어 모델은 만드는 방법을 2가지로 나눌 수 있다.
1. 통계 기반
2. 인공 신경망
2번째 방법을 이용하여 GPT, BERT 모델이 만들어졌다.
</br>

### 언어 모델
언어 모델링 : 주어진 단어들로부터 아직 모르는 단어를 예측하는 작업
언어 모델이 "언어 모델링"을 수행
스탠포드에서는 언어 모델을 grammer로 비유하기도 함
언어 모델이 조합한 단어가 얼마나 적절한지 판단하는 작업이 문법 교정 과정과 비슷하기 때문

### 단어 시퀀스에 확률 할당하기
NLP에서 단어 시퀀스에 확률을 할당하는 것이 왜 필요할까?
"확률"로 문장 구성의 적합도를 판단하기 때문이다.

1. 기계 번역
- ex. P(나는 버스를 탔다) > P(나는 버스를 태운다)
- 언어 모델이 두 문장을 비교해 좌측 문장이 확률적으로 적합하다고 판단

2. 오타 교정(Spell Correction)
- ex. 선생님이 교실로 부리나케 _____
- P(달려갔다) > P(잘려갔다)
